# 離校流程

如有錯誤，請協助[即時回饋]，本手冊按三餐維護更新(笑~)

## 核檢事項

可能沒辦法讓大家打勾勾，但就示意一下。  
各部分詳細步驟，比如論文上傳、裝訂順序、本數等細節，以官方與各系所公布為主。

* [ ] 國內外館際借書及複印資料，均已取件及繳清費用。
* [ ] 還清圖書與繳清滯還金(含本校圖書館、館際借書證單位)
* [ ] 上傳論文電子檔
* [ ] 裝訂紙本論文: 口試委員簽名頁、授權書、論文本體
* [ ] 至圖書館辦理離校並繳交紙本論文
